Personal Boards

=================================
Команды для запуска:
npm run dev - запуск front + back
npm run server - запуск сервера
npm run client - запуск frontend
==================================
Описание функционала:
Входная точка приложения — страница аутентификации;
Пользователи выполняют аутентификацию с помощью email и пароля;
Реализован выход из системы;
Возможность создавать доску;
Добавлять и удалять списки;
Реализована кнопка "Show menu"; 
Перенос карточки из одного списка в другой.
